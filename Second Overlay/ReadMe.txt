How to add the overlay into OBS
-------------------------------
Step 1. Open OBS

Step 2. Go to (Scene Collection > Import > Add)

Step 3. Locate to the file you want and open it, then click the .json 
file and open it

Step 4. Now import it and go back to the scene collection and click the
one you just imported.

Other notes: 

1. You can move all of the widgets and blocks if you want.

2. You can also change things in the config.json in your normal atmoshpericX
folder but just know that it will change all of the customizations if
any update changes files in the config.json.
